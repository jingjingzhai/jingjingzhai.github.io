A transcription factor hierarchy defines an environmental stress response network

Mar 09, 2019Science点击

0. 简介

转录因子（TF）通常一个接一个地研究或者在一些相关因子的集群中研究。然而，转录变化对环境压力响应的整合和网络通常涉及许多相关的TF。在许多生物体中，例如植物，重叠的功能可能使得难以理解如何通过由这些TF控制的复杂信号传导网络实现生物学相关的最终结果。为了更好地了解参考植物拟南芥如何处理通过激素脱落酸（ABA）限制水所引起的压力，作者研究了体内所有与ABA相关的21个TF结合的DNA序列。

1. 摘要

微生物，植物和动物普遍受到环境压力。然而，多细胞生物体中应激反应转录因子（TF）网络的系统研究受到了很大的限制。植物激素脱落酸（ABA）影响数千个基因的表达，使我们能够描述其复杂的应激反应调节网络。作者使用ChIP-Seq测序技术，研究了21个ABA相关TF的全基因组binding，以构建拟南芥的综合调控网络。定义了动态TF结合的决定因素和TF之间的等级，阐明了差异基因表达模式与ABA途径反馈调节之间的关系。通过推断观察到的规范ABA途径组分的调节特征，我们鉴定了一个新的转录调节因子家族调节ABA和盐的反应性，并证明了它们调节植物对渗透胁迫的适应性的效用。

2. 前言

  • 转录是基因表达的关键步骤。已经进行了协同努力来绘制人，苍蝇和蠕虫中的功能元件，包括通过使用染色质免疫沉淀测序（ChIP-seq）分析转录因子（TF）结合所鉴定的大量顺式调节元件。一个尚未开发的领域是刺激如何调节TF结合和随后的转录组变化。此外，与动物研究相比，很少有全面的体内TF结合数据集可用于植物界。为了开始解决这一知识差距，我们生成了100多个ChIP-seq和时间序列RNA测序（RNA-seq）数据集，以表征受刺激影响的转录网络，并绘制参考植物拟南芥中的功能性顺式调控元件。 ，重点是植物激素脱落酸（ABA）。对ABA的反应为检测受刺激影响的转录调节提供了通用模型。 ABA触发数千个基因和TF的差异表达（DE），提供强大的响应，使复杂的基因调控网络建模。此外，ABA在各种植物过程中的作用对基础生物学和农业都很重要；
  • 脱落酸在优化植物的水分利用中起着关键作用，并且是种子发育和对多种环境胁迫（例如干旱和高盐度）的响应所必需的。在拟南芥中，ABA被PYR / PYL / RCAR受体蛋白识别。 ABA的结合触发PYR / PYL / RCAR与A组PP2C蛋白磷酸酶的相互作用并使SnRK2蛋白激酶去阻遏。 SnRK2随后激活诸如TF的底物并引发ABA反应。尽管已知许多TF对ABA反应有贡献，但对它们的靶基因以及这些靶标组合调节的方式知之甚少。体外方法，包括最近描述的拟南芥cistrome数据集，已经能够鉴定DNA基序和相关TF的推断。然而，由于许多大型多成员TF家族，准确的预测仍然具有挑战性。此外，难以在TF结合和转录组变化之间建立直接联系或通过体外测定解决TF调节的动态。因此，我们应用ChIP-seq明确发现TF靶标，检查受刺激影响的TF结合动力学，并将这些发现与随后的转录组变化联系起来。
