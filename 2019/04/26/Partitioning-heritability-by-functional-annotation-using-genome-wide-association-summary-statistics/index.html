Partitioning heritability by functional annotation using genome-wide association summary statistics

Apr 26, 2019Nature Genetics点击

0. 简介

文章链接

1. 摘要

  • 最近的研究工作已经证明了基因组中一些功能分类 (functional categories)对复杂疾病的遗传力有不同的贡献，作者在这里分析了各种各样的功能元件，包括：细胞类型特异性的元件，来估计它们在17个复杂疾病的GWAS结果中 (平均样本量73599) 对遗传力的贡献；
  • 为了达到此目的，作者引入了一个新方法，即: stratified LD score regression, 从GWAS summary statistics并考虑连锁的标记划分遗传力，这个新方法即使对于大样本量也是可行的，而且利用了全基因组的信息；
  • 作者发现：
    • 很多性状中，保守区域富集很大的遗传力；
    • 在FANTOM5增强子区域有免疫疾病特异性的遗传力的富集；
    • 很多细胞类型特异性的富集包括：BMI、age at menarche、受教育程度以及抽烟行为等性状的遗传力在中枢神经系统有显著地富集。

2. 前言

  • 对于复杂性状的GWAS来说，很多遗传力都蕴含在那些没有达到GWAS显著性水平的SNP中 (主要还是样本量的问题)；然而，目前有很多方法尝试利用功能信息和GWAS数据（使用GWAS显著地位点）推断疾病生物学，没有考虑到连锁不平衡的信息。本文的目标是通过使用所有SNP的信息以及专门的模拟LD来估计可归因于各种各样功能分类的全基因组范围内SNP遗传力 (原话：We aim to improve power by estimating the proportion of genome-wide SNP heritability1 attributable to various functional categories, using information from all SNPs and explicitly modeling LD)；
  • 之前的工作用于划分SNP遗传力是GCTA软件中的REML(Restricted maximum likelihood)方法, REML需要个体的基因型，但是很多大规模的GWAS分析是基于meta分析，因此只有summary statistics没有但各个体的基因型。而且即使有个体水平的基因型，REML如果样本量大话计算上也很困难。因此，作者在这里介绍了用于划分遗传力的方法，只需要GWAS summary statistics，从reference panel获得的LD信息即可；
